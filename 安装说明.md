# AI Chat Width Controller - 安装说明

## 📋 安装前准备

### 1️⃣ 生成图标文件（三选一）

扩展需要图标文件才能正常显示。选择以下任一方法：

#### 方法 A：使用 Python 脚本（推荐）
```bash
# 安装依赖
pip install Pillow

# 运行脚本生成图标
python generate-icons.py
```

#### 方法 B：使用浏览器生成
1. 用浏览器打开 `create-icons.html`
2. 图标会自动下载到下载文件夹
3. 将下载的 `icon16.png`, `icon48.png`, `icon128.png` 移动到 `icons/` 文件夹

#### 方法 C：使用在线工具
1. 访问 https://www.favicon-generator.org/
2. 上传图片或生成图标
3. 下载 16x16, 48x48, 128x128 三个尺寸
4. 重命名为 `icon16.png`, `icon48.png`, `icon128.png`
5. 放入 `icons/` 文件夹

---

## 🚀 安装扩展

### Chrome 浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的「**开发者模式**」开关
4. 点击「**加载已解压的扩展程序**」按钮
5. 选择 `wide-ai-page` 文件夹
6. 安装完成！扩展图标会出现在工具栏

### Edge 浏览器

1. 打开 Edge 浏览器
2. 在地址栏输入：`edge://extensions/`
3. 打开左下角的「**开发人员模式**」开关
4. 点击「**加载解压缩的扩展**」按钮
5. 选择 `wide-ai-page` 文件夹
6. 安装完成！扩展图标会出现在工具栏

## ✅ 验证安装

1. 访问支持的网站（如 https://chat.deepseek.com）
2. 点击浏览器工具栏中的扩展图标

## 🎯 使用技巧

### 快速切换
- 点击扩展图标即可快速切换宽度
- 设置会自动保存，下次访问自动应用

### 推荐设置
- **大屏幕（≥1920px）**：使用「全宽模式」
- **中等屏幕（1440-1920px）**：使用「较宽模式」
- **小屏幕（<1440px）**：使用「原始宽度」

### 支持的网站

## ❓ 常见问题

### Q: 点击扩展图标没有反应？
**A:** 确保图标文件已正确生成并放置在 `icons/` 文件夹中。

### Q: 切换宽度后没有效果？
**A:** 
1. 确认当前网站在支持列表中
2. 尝试刷新页面（F5）
3. 重新加载扩展

### Q: 如何卸载？
**A:** 
1. 进入 `chrome://extensions/` 或 `edge://extensions/`
2. 找到「AI Chat Width Controller」
3. 点击「移除」按钮

### Q: 如何更新扩展？
**A:** 
1. 修改代码后，进入扩展管理页面
2. 点击扩展卡片上的「刷新」图标
3. 刷新正在使用的网页

## 🔧 故障排除

### 扩展无法加载
- 检查文件夹结构是否完整
- 确保 `manifest.json` 文件存在且格式正确
- 查看浏览器控制台的错误信息

### 样式不生效
- 打开浏览器开发者工具（F12）
- 检查 Console 标签页是否有错误
- 确认 `content.js` 和 `content.css` 已正确加载

### 图标显示异常
- 确保 `icons/` 文件夹中有三个 PNG 文件
- 文件名必须完全匹配：`icon16.png`, `icon48.png`, `icon128.png`
- 检查文件是否损坏
