图标文件说明
==============

由于无法直接生成 PNG 文件，您需要手动创建图标文件。

方法一：使用在线工具
1. 访问 https://www.favicon-generator.org/ 或类似网站
2. 上传一个图片或使用文字生成图标
3. 下载 16x16, 48x48, 128x128 三个尺寸的 PNG 文件
4. 重命名为 icon16.png, icon48.png, icon128.png
5. 放置在此 icons 文件夹中

方法二：使用 create-icons.html
1. 在浏览器中打开项目根目录的 create-icons.html
2. 会自动下载三个图标文件
3. 将下载的文件移动到此 icons 文件夹

方法三：使用任意图片
1. 准备一张正方形图片
2. 使用图片编辑软件（如 Paint、Photoshop）调整为 16x16, 48x48, 128x128 三个尺寸
3. 保存为 PNG 格式
4. 重命名并放置在此文件夹

临时方案：
如果暂时没有图标，可以使用任意 PNG 图片，扩展功能不受影响。

建议图标设计：
- 使用紫色渐变背景 (#667eea 到 #764ba2)
- 中间显示宽度调整的符号（如双向箭头或三条横线）
- 保持简洁，确保在小尺寸下清晰可见
