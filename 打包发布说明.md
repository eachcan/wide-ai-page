# 扩展打包发布说明

## 📦 打包步骤

### 方法一：手动打包（推荐）

1. **创建发布文件夹**
   ```
   创建一个新文件夹，例如：ai-chat-width-controller-release
   ```

2. **复制必要文件**（只复制以下文件）
   - `manifest.json`
   - `popup.html`
   - `popup.js`
   - `content.js`
   - `content.css`
   - `background.js`
   - `icons/` 文件夹（包含所有 PNG 文件）
   - `README.md`（可选，用于说明）

3. **压缩成 ZIP**
   - 选中上述所有文件和文件夹
   - 右键 → 发送到 → 压缩(zipped)文件夹
   - 或使用 7-Zip、WinRAR 等工具压缩
   - 命名为：`ai-chat-width-controller-v1.0.0.zip`

### 方法二：使用命令行打包

运行以下命令自动打包：

```powershell
# 在项目根目录执行
.\pack-extension.ps1
```

### 2. 发布流程

1. **注册开发者账号**
   - 访问：https://chrome.google.com/webstore/devconsole
   - 支付一次性注册费：$5 USD
   - 填写开发者信息

2. **上传扩展**
   - 点击「新增项目」
   - 上传 ZIP 文件
   - 等待自动验证

3. **填写商店信息**
   - **名称**：AI Chat Width Controller
   - **简短描述**：一键调整 AI 对话网站宽度，支持 DeepSeek、豆包、元宝、ChatGPT、Gemini
   - **详细描述**：参考下方模板
   - **类别**：生产力工具
   - **语言**：中文、英文

4. **上传图片资源**
   - 图标：使用 `icons/icon128.png`
   - 截图：建议制作 3-5 张展示效果的截图
   - 宣传图：可选

5. **隐私设置**
   - 声明不收集用户数据
   - 填写隐私政策（如果有网站）

6. **提交审核**
   - 检查所有信息
   - 点击「提交审核」
   - 等待 1-3 个工作日

## ⚠️ 注意事项

1. **ZIP 文件结构**：确保文件在 ZIP 根目录，不要有额外的文件夹层级
2. **文件大小**：保持在 10MB 以下（当前约 20KB）
3. **权限说明**：manifest.json 中的权限要合理，避免申请不必要的权限
4. **测试**：发布前在本地充分测试所有支持的网站
5. **更新频率**：网站更新可能导致选择器失效，需要及时维护
