# 🚀 快速开始

## 第一步：生成图标

**最简单的方法 - 使用浏览器：**

1. 双击打开 `create-icons.html` 文件
2. 浏览器会自动下载 3 个图标文件到下载文件夹
3. 将这 3 个文件移动到 `icons/` 文件夹中

完成！

---

## 第二步：安装扩展

### Chrome 用户：
1. 打开 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `wide-ai-page` 文件夹

### Edge 用户：
1. 打开 `edge://extensions/`
2. 开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择 `wide-ai-page` 文件夹

---

## 第三步：开始使用

1. 访问 ChatGPT、DeepSeek、豆包等网站
2. 点击浏览器工具栏的扩展图标
3. 选择您想要的宽度模式
4. 享受更宽广的阅读体验！

---

## ⚡ 常见问题

**Q: 图标在哪里？**  
A: 需要先生成。打开 `create-icons.html` 自动下载。

**Q: 扩展无法加载？**  
A: 确保 `icons/` 文件夹中有三个 PNG 文件。

**Q: 切换后没效果？**  
A: 刷新页面（F5）后重试。

**Q: 支持哪些网站？**  
A: DeepSeek、豆包、元宝、ChatGPT、Gemini
